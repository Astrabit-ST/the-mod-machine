RPG::CommonEvent

Data class for common events.

Superclass

  • Object

Attributes

id

The event ID.

name

The event name.

trigger

The event trigger (0: none, 1: autorun; 2: parallel).

switch_id

The condition switch ID.

list

List of event commands. An RPG::EventCommand array.
